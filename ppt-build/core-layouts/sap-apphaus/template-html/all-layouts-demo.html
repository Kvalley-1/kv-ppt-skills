C · Introduction — 雙欄純文字開場

INTRODUCTION

這套課不是教你「Claude Code 怎麼用」。是教你怎麼讓 AI 替你工作——把你從操作員升級成主管。

過去一年，AI 從「玩具」變成「員工」。會用的人產出爆增 5-10 倍，不會用的人還在自己查資料、自己寫文件。差距會持續拉開。

智谷的同仁人均已配 Claude Code Pro，但大部分人只用了 5%。問題不在工具，在心態——你還用「自己做事順便問 AI」的方式，不是「下指令讓 AI 做事自己審核」。

這套課帶你完成這個切換。三章節：心態轉換 → 提問框架 → 實戰練習。學完後，你會發現過去一週的工作量，AI 兩小時就能做完。

D · Detail — 主內容 + 側欄案例（紫色主題）

CORE MINDSET
心態轉換 / Mindset Shift

真正能用好 AI 的人，把自己當主管，不是操作員。你的價值在判斷跟決策，不在執行。

能讓 AI 做的事，絕對不要自己動手：

  • 「你去查 [主題]」— 取代自己 Google
  • 「你去裝 [工具]」— 取代自己看安裝文件
  • 「你去研究 [選項]」— 取代自己讀比較文章

這週你還有哪些工作，
是 AI 可以替你做但你還在自己做的？

EXAMPLE MOVES
把這三句話掛在嘴邊，立刻換腦袋。
指派
「你去查 SAP S/4HANA 跟 Oracle ERP 的核心差異，列 5 點。」
委任
「你去裝 Playwright，設好我的測試環境。」
要求觀點
「你去研究 3 種雲端方案，告訴我推哪個、為什麼。」
