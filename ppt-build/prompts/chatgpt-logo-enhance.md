# ChatGPT / GPT-4o Image Edit — Logo 修圖 Prompt 範本

> 當客戶 logo 模糊 / 太小 / 有底色，且 Claude 無法處理時，給 Jimmy 這個 prompt 帶到 ChatGPT/GPT-4o 修圖。

---

## 🔴 核心鐵律

**禁止 AI 重畫 logo**。所有修圖只能 enhance，不能重設計。

---

## Prompt 範本（複製給 ChatGPT / GPT-4o）

```
我有一張 [客戶名稱] 的 logo，目前狀況是：[勾選]
- [ ] 解析度太低（< 300px）
- [ ] 邊緣模糊 / 鋸齒
- [ ] 顏色失真 / 偏色
- [ ] 有白色 / 灰色 / 其他底色（要去背）
- [ ] 周圍有多餘空白要裁切

請幫我做以下處理：

1. **保持原本設計** ⚠️ 重要：不要重畫、不要簡化、不要修改任何元素，只能 enhance
2. **提高解析度**到 1024×1024（保持比例）
3. **透明背景**（PNG 格式）
4. **顏色精確**（保持原本品牌色，不要色偏）
5. **邊緣銳利**（消除鋸齒，但保持原本筆畫粗細）
6. **置中**（logo 在 1024×1024 畫布中央）

輸出：單一 PNG 檔案，透明背景。

⚠️ 如果你判斷需要任何「重新設計」才能達到上述要求，請停止並告訴我，我會用其他方式處理。不要擅自重畫。
```

---

## 不同情境的子 prompt

### 情境 A：只是模糊（其他都 OK）

```
這張 [客戶] logo 邊緣模糊，請幫我：
1. 銳利化邊緣
2. 保持原本所有元素、顏色、比例
3. 輸出 1024×1024 PNG，透明背景
4. ⚠️ 禁止重畫
```

### 情境 B：太小要放大

```
這張 [客戶] logo 只有 200×200px 太小，請幫我：
1. 用 AI upscale 到 1024×1024
2. 保持原本設計（向量質感）
3. 透明背景
4. ⚠️ 禁止重設計，只是放大
```

### 情境 C：有底色要去背

```
這張 [客戶] logo 有白色背景，請幫我：
1. 去背（透明背景）
2. 邊緣抗鋸齒處理（避免去背後出現生硬邊緣）
3. 不要修改 logo 本身
4. 輸出 PNG
```

### 情境 D：顏色偏色

```
這張 [客戶] logo 顏色看起來偏色（看起來不像官方版本），請幫我：
1. 校正顏色到接近 [客戶] 官方品牌色（[填色票，例如 #0066CC]）
2. 不要修改設計、字體、形狀
3. 透明背景輸出
4. ⚠️ 只調整色相 / 飽和度，不要重畫
```

---

## 收到回傳後的檢查清單

Jimmy 拿到 ChatGPT/GPT-4o 輸出後：
- [ ] Logo 設計完全沒被改（比對原圖）
- [ ] 顏色正確
- [ ] 解析度足夠（≥ 500px 寬）
- [ ] 透明背景（檢查邊緣沒白邊）
- [ ] 邊緣銳利

通過 → 存到 `knowledge/clients/assets/<客戶>/logo.png`

---

## 萬一 ChatGPT 還是重畫了

回覆：「你重畫了 logo，這違反我的指示。請停止並告訴我為什麼無法 enhance。我用其他工具處理。」

通常表示：
- 原圖品質實在太差，AI 無法 enhance 只能重畫
- → 改用：找官網更高解析度版本 / 客戶提供印刷檔 / Adobe Illustrator AI Vector Trace
